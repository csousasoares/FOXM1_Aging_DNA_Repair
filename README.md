# FOXM1 enhances DNA repair in aged cells to maintain the peripheral heterochromatin barrier against senescence enhancers


Repository for the paper "FOXM1 enhances DNA repair in aged cells to maintain the peripheral heterochromatin barrier against senescence enhancers" (https://doi.org/10.1101/2025.10.29.685369).

## RNA-seq Analysis

aging_fibroblasts_analysis - Code for differential expression analysis and GSEA between older and younger cohorts of fibroblasts, and heatmap of the FOXM1 DNA Repair network expression across age groups.

dna_repair_gsea_foxm1 - GSEA of FOXM1 knockdown or overexpression and PCA using DNA repair genes from Biological Processes

pca_and_volcano_foxm1_data - Volcano plot of FOXM1 knockdown or overexpression and PCA os differential gene expression analysis.

## H3K9me2 Immunofluorescence or CUT&RUN Analysis

h3k9me2_cut_run_regions - Analysis of differentially bound H3K9me2 regions (DBRs) using DiffBind, genomic features of DBRs using ChIPSeeker, rGREAT analysis and motif-distance matrix analysis.

h3k9me2_g9a - Correlation between G9a and H3K9me2 protein levels at single cell level inferred from immunofluorescence.
